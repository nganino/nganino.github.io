About Me

I am an Electrical Engineering student at Purdue University focusing on image processing, optics, computer vision, and semiconductor manufacturing. I conduct undergraduate research in image formation and depth-from-defocus systems, with experience in Fourier optics and computational imaging.

In Summer 2025 I will be working at TSMC as an Equipment Engineer. My interests span imaging systems, optical hardware, sensor physics, and embedded system design.

Research

Accelerated Point Spread Function Computation

I am developing a fast method for computing PSFs using Fourier-domain decomposition and structured optimization, enabling significant speedup for imaging simulation pipelines.

Preprint coming December 2025.

Depth-from-Defocus Image Processing

Researching depth estimation using blur gradients and frequency-domain cues, with applications in compact imaging systems and computational photography.

Fourier Optics 4f System for Edge Detection

Built a physical 4f optical processor to demonstrate spatial filtering and real-time edge detection using Fourier masks.

Projects

Plasma Speaker System

High-voltage audio plasma speaker using MOSFET modulation and custom wound transformer.

ESP32-S3 LCD + FFT Analyzer

Real-time FFT visualization on a TFT LCD using the ESP32-S3 and optimized DMA pipelines.

Image Sensor Analysis Tools

Python and C tools for exploring noise, pixel statistics, and radial PSF functions.
